# Ledger

Ledger is a market companion designed to provide institutional-grade stock research. Ledger helps you identify structural opportunities, assess risks, and track market momentum.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   Create a `.env` file in the root directory and add your required API keys.

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
